# f2e-3rd-tour-guide

>F2E-3rd參賽作品
>
>TDX API - (Transport Data eXchange)

## 簡介

* 台灣觀光網站，可查詢景點、美食、住宿、活動
* [Demo](http://dobi8422.github.io/f2e-3rd-tour-guide/)

## 功能

* 關鍵字搜尋
* 篩選功能(多選)(按鈕:景點分類，縣市分類)
* mobile/iPad/desktop介面
* 調整閱讀模式(圖文/列表) (for iPad/desktop)
